About WildLens AI

Built by students, powered by science — dedicated to protecting the world's most vulnerable species.

AI in service
of nature.

WildLens AI is a generative AI system designed to support wildlife conservation efforts. By automating species identification from camera trap photographs, we help researchers and forest departments monitor endangered populations without manual image analysis.

Built as a collaborative open-source project, the system uses ResNet50 and EfficientNet models trained on the iNaturalist and COCO datasets, combined with Claude AI for automated conservation report generation.

// Technology Stack

Built with cutting-edge tools

Every component chosen for performance, scalability, and accuracy.

⚛️
React.js + Vite
Lightning-fast frontend with component-based architecture and hot module replacement.
🐍
FastAPI (Python)
Asynchronous REST API backend with automatic documentation and type safety.
🔥
Node.js + Firestore
Real-time NoSQL database microservice for storing all sighting and species data.
🧠
PyTorch + ResNet50
Transfer learning on pre-trained ImageNet models fine-tuned for wildlife classification.
🎯
YOLOv8
Real-time object detection for locating and drawing bounding boxes around animals.
🤖
Anthropic Claude
Generative AI for writing professional, context-aware wildlife conservation reports.
☁️
Cloudinary
Cloud image storage and delivery with automatic optimisation and CDN distribution.
📦
iNaturalist Dataset
3M+ labeled wildlife images across 10,000+ species used for model training.
🐙
GitHub
Collaborative development with branch-based workflow. Repository: raz-88/wildlife-ai.
